## Setting Up
### Requirements
Python 3.9 or higher<br>
Pip 3<br>
Sqlite3<br>

### Step 1 :- Clone the repository
git clone <repo_url>

### Step 2 :- Create virtual environment
$ pip3 install virtualenv
$ virtualenv env

### Step 3 :- Install packages
$ pip3 install flask
$ pip3 install flask_wtf
$ pip3 install PyPDF2

### Step 4 :- Run the Flask application
$ python app.py
